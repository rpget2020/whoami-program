# whoami-program

이병욱 백엔드 개발자 경력·자기소개 포트폴리오 사이트입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

## 빌드

```bash
npm run build
npm run preview
```

## 구성

- **Hero** — 소개 및 핵심 기술 태그
- **About** — 성장 과정, 강점/약점, 목표 및 비전
- **Experience** — 경력 사항
- **Projects** — 주요 프로젝트 5건 (전자서명, CSAP, 문서 편집, 화면 공유, 리포트 컨버터)
- **Skills** — 기술 스택
- **Education** — 학력 및 자격증
- **Contact** — 연락처

내용 수정은 `src/data/resume.js` 파일에서 할 수 있습니다.
